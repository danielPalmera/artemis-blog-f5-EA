# Artemis Blog

Monorepo con API en Express (`backend`) y un frontend estático mínimo (`frontend/index.html`).

## Comandos

| Comando           | Acción                                              |
| :---------------- | :-------------------------------------------------- |
| `pnpm install`    | Instala dependencias                              |
| `pnpm dev`        | Inicia API (3000) y frontend (4321) en paralelo     |
| `pnpm dev:api`    | Solo la API en el puerto 3000                       |
| `pnpm dev:front`  | Solo el frontend en http://localhost:4321           |
| `pnpm build`      | Compila el backend                                  |
| `pnpm start`      | Ejecuta el backend compilado                        |
